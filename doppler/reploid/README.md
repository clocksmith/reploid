# REPLOID

**Recursive Evolution Protocol Loop Orchestrating Inference DOPPLER**

Browser-native sandbox for safe AI agent development with self-modification capabilities.

**[Try it live at replo.id/r](https://replo.id/r)**

See the main [README](../../README.md) for full documentation.

## Quick Start

```bash
npm install
npm run serve     # Landing page + /r (Reploid) + /d (Doppler) at :8080
npm run dev       # Full dev server with API proxies at :8000
npm run doppler   # Doppler-only mode at :8080
```

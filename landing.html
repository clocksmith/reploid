Three Levels of Self-Improvement

REPLOID doesn't just execute code—it creates tools, improves those tools, then improves how it creates tools.

LEVEL 1 RSI

Creates Tools at Runtime

Agent writes new functions and saves them to the Virtual File System. These tools persist and can be called in future iterations.

Example: Agent creates analyze_code.js to parse JavaScript files.
LEVEL 2 RSI

Improves Tool Creation Mechanism

Agent creates meta-tools that create tools. It modifies its own tool-writing process to become more efficient.

Example: Agent builds create_tool_from_description.js (a tool that writes tools).
LEVEL 3 RSI

Modifies Core Substrate

Agent loads new modules into its own cognitive loop. It can replace core systems like the tool-runner or agent-loop itself.

Example: Agent rewrites agent-loop.js to add parallel tool execution.

What is REPLOID?

REPLOID is a browser-native agentic AI system designed for Recursive Self-Improvement (RSI) via source code manipulation. The core hypothesis: the browser is the perfect ecosystem for an RSI agent.

Why the browser? It provides rich Web APIs for visualization, built-in sandboxing for safe code execution, visual feedback loops for better human-AI collaboration, and universal deployment with no installation required.

REPLOID operates with human-in-the-loop approvals at critical decision points, ensuring you maintain full control over all changes including changes to its own source code. The system uses a structured workflow with context curation and explicit change proposals that require your approval before being applied.

Agent Workflow

REPLOID's DREAMER engine follows a structured 8-state workflow:

  • IDLE ‚Üí Set a goal to begin
  • CURATING_CONTEXT ‚Üí Agent selects relevant files
  • AWAITING_CONTEXT_APPROVAL ‚Üí Review and approve context bundle
  • PLANNING_WITH_CONTEXT ‚Üí Agent analyzes and plans changes
  • GENERATING_PROPOSAL ‚Üí Creates dogs.md change bundle
  • AWAITING_PROPOSAL_APPROVAL ‚Üí Interactive diff review with selective approval
  • APPLYING_CHANGES ‚Üí Applies approved changes with checkpoint
  • REFLECTING ‚Üí Learns from outcome for future improvements
